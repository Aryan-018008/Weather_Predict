# About
🌤 WeatherFore — Weather Forecast Web App

WeatherFore is a modern weather forecasting web application that provides real-time weather updates and a 5-day forecast for any location using the OpenWeather API.

Built using HTML, CSS, and JavaScript, the app focuses on clean UI, responsive design, and dynamic weather data fetching.

#  Live Demo

👉 https://weatherforecasting008.netlify.app/


 # Features

✅ Real-time weather information
✅ 5-day weather forecast
✅ City-based search
✅ Dynamic background based on weather
✅ Dark mode toggle 🌙
✅ Live date & time display
✅ Responsive UI design
✅ Refresh weather button

 # 🔑 API Setup (Important)

This project uses the OpenWeather API.

- Create account at:
https://openweathermap.org/api

- Generate API Key.

- Add your API key inside script.js:

- const apiKey = "YOUR_API_KEY";

⚠️ Do NOT expose API keys in public repositories.

# 🌍 How It Works

- User enters a city name.
- App sends request to OpenWeather API.
- Weather data is fetched using fetch() and async/await.
- UI updates dynamically with:
- Temperature
- Humidity
- Wind Speed
- Weather Condition
- Forecast cards render automatically.
